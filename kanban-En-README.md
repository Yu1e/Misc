## üöÄ Features

*   **Zero Dependencies:** Built using Vanilla JavaScript, HTML5, and CSS3. No libraries, frameworks, or build steps required.
*   **Single File Deployment:** The entire application lives in `kanban.html`. Just download and run.
*   **Drag & Drop Interface:** Intuitive drag-and-drop functionality to move tasks between columns.
*   **Persistent Data:** Uses the browser's **Local Storage** to save your tasks. Your data survives page refreshes and browser restarts without a backend server.
*   **CRUD Operations:** Easily **C**reate, **R**ead, **U**pdate, and **D**elete tasks.
*   **Responsive Design:** Works neatly on different screen sizes.

## üõ†Ô∏è How to Use

### Installation
Since this is a single-file application, "installation" is instant:

1.  Download the `kanban.html` file from this repository.
2.  Open the file in any modern web browser (Chrome, Firefox, Safari, Edge).

### Usage
1.  **Add a Task:** Click the "+" button (or "Add Item") in a column to create a new card.
2.  **Move a Task:** Click and hold a card, then drag it to the desired column (e.g., from "To Do" to "Done").
3.  **Edit/Delete:** Click on a task to edit the text or remove it from the board.
4.  **Save:** Data saves automatically to your browser's storage.
